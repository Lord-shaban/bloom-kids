# 🌱 Bloom Kids — الموقع الرسمي

موقع لاندينج بيدج احترافي لـ **بلوم كيدز** — سيشنز أونلاين تفاعلية للأطفال من 6 إلى 12 سنة.

موقع ستاتيك خالص (HTML + CSS + JS) — من غير أي build step — جاهز للرفع مباشرة على **Cloudflare Pages**.

---

## ⚠️ قبل ما ترفع الموقع — 3 حاجات لازم تتظبط

### 1. رقم الواتساب (الأهم!)
افتح ملف [js/main.js](js/main.js) وغيّر الرقم في أول سطر:

```js
const WHATSAPP_NUMBER = "201000000000"; // ← حط رقمكم هنا
```

الصيغة: كود الدولة + الرقم **من غير** صفر في الأول ومن غير `+`
مثال: الرقم المصري `01012345678` يتكتب `201012345678`

كل أزرار الحجز والأيقونة العائمة بتفتح واتساب على الرقم ده برسالة جاهزة ببيانات الحجز.

### 2. لينكات السوشيال ميديا
في ملف [index.html](index.html) — دوّر على كلمة `socials` وحدّث لينكات فيسبوك وانستجرام بحساباتكم الحقيقية.

### 3. آراء الأهالي
قسم "كلام الأهالي" فيه أمثلة توضيحية مؤقتة — **استبدلوها بآراء حقيقية** من أولياء أمور مشتركين قبل الإطلاق.

---

## 🚀 الرفع على Cloudflare Pages

### الطريقة الأسهل (Drag & Drop):
1. ادخل على [dash.cloudflare.com](https://dash.cloudflare.com)
2. من القائمة: **Workers & Pages** ← **Create** ← **Pages** ← **Upload assets**
3. اكتب اسم للمشروع (مثلًا `bloom-kids`)
4. اسحب محتويات المجلد ده كله (الملفات نفسها، مش المجلد) وارفعها
5. اضغط **Deploy** — وخلاص! الموقع هيبقى شغال على `bloom-kids.pages.dev`

### الطريقة الاحترافية (Git):
1. ارفع المشروع على GitHub
2. في Cloudflare Pages اختار **Connect to Git**
3. اختار الريبو — ومن غير أي إعدادات build (سيبها فاضية)
4. كل push جديد هيتنشر أوتوماتيك

### ربط دومين خاص:
من إعدادات المشروع في Cloudflare Pages ← **Custom domains** ← ضيف الدومين بتاعكم (مثلًا `bloomkids.com`).

## 📁 هيكل المشروع

```
bloom kids/
├── index.html          ← الصفحة الرئيسية (كل الأقسام)
├── 404.html            ← صفحة الخطأ
├── _headers            ← إعدادات الكاش والأمان لـ Cloudflare
├── robots.txt          ← إعدادات محركات البحث
├── css/
│   └── style.css       ← كل التنسيقات (ألوان الهوية في أول الملف)
├── js/
│   └── main.js         ← التفاعلات + رقم الواتساب
├── assets/
│   └── images/         ← الصور المستخدمة في الموقع
└── image/              ← الصور الأصلية (مش مستخدمة في الموقع — ممكن متترفعش)
```

## 🎨 تعديل الألوان

كل ألوان الهوية متعرّفة كمتغيرات في أول ملف [css/style.css](css/style.css):

```css
:root {
  --navy: #332d6e;    /* الكحلي */
  --purple: #6a4fb6;  /* البنفسجي */
  --teal: #23a89f;    /* التركواز */
  --pink: #ef5f7c;    /* الوردي */
  --yellow: #f6a92c;  /* الأصفر */
  --green: #7dc242;   /* الأخضر */
  --cream: #fdf6ec;   /* الخلفية الكريمي */
}
```

غيّر أي لون من هنا وهيتغير في الموقع كله.
